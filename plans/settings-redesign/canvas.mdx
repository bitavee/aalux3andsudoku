<Annotation id="n-main-reader" title="Within-tab grouping" targetId="a-main-reader" placement="bottom">

The Reader category has ~14 items. Light section rules (TEXT / LAYOUT / SCREEN) break the wall without adding a navigation level. Focused value renders as a black stepper pill `‹ Medium ›` (reuses Lyra's `highlightValue` chip). Right edge keeps Lyra's real scrollbar.

</Annotation>
<Annotation id="n-main-system" title="Actions read differently from values" targetId="a-main-system" placement="bottom">

Drill-in actions get a leading icon tile + trailing chevron ("opens a screen"); inline values/toggles do not. Toggles render as iOS-style switches (knob-right + filled track = on). Icon tiles map to Lyra's existing 24px bitmap set.

</Annotation>
